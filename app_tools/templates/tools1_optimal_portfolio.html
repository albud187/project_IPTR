This tool takes historical data from each ticker between the start and end dates, various portfolio allocations and returns the allocation with the best sharpe ratio over the time period

Start/End Date: Enter date in the format YYYY,MM,DD

Portfolios to Simulate: Enter the number of portfolios to simulate. For example, simulating 1000 portfolios is sufficient for 4 tickers.
More simulations may be needed for portfolios with more tickers.
