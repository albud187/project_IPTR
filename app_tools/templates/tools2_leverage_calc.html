Leverage Calculator


Lever to Target Return


Lever to Risk Tolerance

This tool takes in a portfolio, and using historical returns / risk between the start and end date calculates the leverage needed based on a target return or an acceptable risk tolerance

Portfolio Holdings: list of tickers and their percent of capital allocation in the form ticker1:allocation1,ticker2:allocation2,ticker3:allocation3... (no space).
Example: SPY:30,QQQ:35,IWM:25,GDX:10
note: holdings must be positive and must add to 100

Interest Rate: Annual interest rate to borrow money expressed as a decimal.
Example: 0.015 corresponds to 1.5% interest rate

Start / End Date: Date in format YYYY,MM,DD.

Target Return: Target expected return expressed as a decimal.
Example: 0.11 corresponds to 11% expected annual returns

Risk Tolerance: Expected annual standard deviation of returns expressed as a decimal
Example: 0.09 corresponds to a 9% standard deviation of expected returns.
note: must be positive number for result to make sense
