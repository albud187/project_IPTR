Correlation Matrix Calculator



This tool returns a table listing the correlations between the input tickers and the input portfolio.

Tickers: Input a list of tickers to exampine for correlations in the form ticker1, ticker2, ticker 3.
For example: QQQ, IWM, SPY

Holdings: list of tickers and their percent of capital allocation in the form ticker1:allocation1,ticker2:allocation2,ticker3:allocation3... (no space).
Example: SPY:30,QQQ:35,IWM:25,GDX:10

Start/End Date: Date in format YYYY,MM,DD.
